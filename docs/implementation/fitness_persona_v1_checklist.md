### Acceptance Criteria
**Daily Context**
- Readiness Snapshot accurately reflects calculated daily state
- Sleep Trend graph toggles correctly between 7, 14, and 30 day views
- Energy Trend graph toggles correctly between 7, 14, and 30 day views

**Action Cards**
- Users can pin/unpin routines to the top
- Habits are visually stacked by category
- Sort order adapts based on "last week vs this week" context
- "Complete" button simply marks item done
- "Complete + Log Habits" triggers the detailed logging flow

**Keep**
- Activity Map renders user's history visually
- Goals at a Glance prominently features upcoming dates
- Quick Log allows rapid entry similar to Gratitude Jar

**General**
- Seed data updated per P2 specs
- UI matches Fitness Persona design mocks
